CSS ANIMATIONS

An animation allows an element to gradually change from one style to another With CSS animations, you can changeas many css properties as you wish, as many time as you wish.

The Css proprietes are supported by most browsers like Google chrome, internet explorer, opera mini
    These are the CSS animations proprietes:
  • @keyframes
  • animation-name
  • animation-duration
  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-timing-function
  • animation-fill-mode
